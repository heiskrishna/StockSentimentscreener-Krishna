## 💻 How to Run

1. Clone the repo or download as ZIP
2. Install dependencies:
```bash
pip install -r requirements.txt
```
3. Add your API keys in a `.env` file:
```
TWITTER_BEARER_TOKEN=your_token
NEWS_API_KEY=your_token
```
4. Run the app:
```bash
python app.py
```
